# Users List

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Open-2ea44f?logo=github)](https://madmaxim22.github.io/users-list/)

Небольшое React-приложение для отображения списка пользователей и просмотра детальной информации по выбранному пользователю.

## Возможности

- Загрузка списка пользователей с удаленного API.
- Выбор пользователя из списка и отображение карточки с деталями.
- Состояния загрузки и ошибок для списка и карточки деталей.
- Кнопка повторного запроса (`Retry`) при ошибках.
- Отмена предыдущего запроса деталей пользователя при быстром переключении.

## Технологии

- React
- TypeScript
- Vite
- ESLint

## Запуск проекта

```bash
npm install
npm run dev
```

После запуска приложение будет доступно в браузере по адресу, который покажет Vite (обычно `http://localhost:5173`).

## Доступные скрипты

- `npm run dev` - запуск проекта в режиме разработки.
- `npm run build` - сборка проекта для production.
- `npm run preview` - локальный просмотр production-сборки.
- `npm run lint` - проверка кода линтером.

## Структура проекта

- `src/api` - функции работы с API и обработка ошибок.
- `src/hooks` - кастомные хуки загрузки списка и деталей пользователя.
- `src/components` - UI-компоненты списка и карточки деталей.
- `src/App.tsx` - композиция основных блоков интерфейса.
